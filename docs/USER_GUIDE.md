## ğŸ¯ Making a Schedule (3 Simple Steps)

### Step 1: Select Your Week

![Date Selection](Main%20UI.png)

1. **Click the date field** (shows 11/17/2025)
2. **Pick a Monday** from the calendar
3. The week will automatically start from that Monday

---

### Step 2: Load Events

1. **Click "Fetch Week"** (big blue button)
2. Wait 3-5 seconds while events load from calendar
3. Event cards will appear below

---

### Step 3: Assign Staff

![Event Card Example](Main%20UI.png)

**For each event:**

1. **Review event details:**
   - Date/time shown at top
   - Event name in large text
   - Location (if available)

2. **Assign staff members:**
   - Click employee dropdown
   - Select a person
   - Click **+ Add Staff** to add more people
   - Each person can have different hours

3. **Optional: Add notes**
   - Special instructions
   - Setup requirements
   - Client preferences

4. **Click "Save Assignments"** at bottom of card

5. **Repeat for each event**

## âš¡ Quick Tips

### Assigning Multiple Staff:
1. Select first person from dropdown
2. Click **+ Add Staff**
3. Select next person
4. Repeat as needed
5. Each person can have different times

### Individual Time Slots:
By default, everyone gets the event's full hours (11:30 AM - 02:00 PM).

**To customize:**
- Change the time fields for each person
- Example: One person 11:30-1:00, another 1:00-2:00

### Adding Locations:
If event shows "No location":
1. Click **+ Add Location**
2. Enter venue address
3. It's saved for future events at this venue

### Event Notes:
Add special instructions like:
- "Setup required at 10am"
- "VIP client - extra attention"
- "Load in through back entrance"

## ğŸš¨ Common Issues

### "No events this week"
- The calendar might not have events for this week
- Try selecting a different Monday
- Contact admin if calendar should have events

### Employee not in dropdown
- Person needs to be added to system by admin
- Contact admin with employee name and phone number

### Can't save assignment
- Make sure you selected an employee from dropdown
- Check that start time is before end time
- Verify you clicked the blue "Save Assignments" button

### Location not showing
- Calendar event might not have location info
- Click **+ Add Location** to add it manually
- Admin can also add venues in the system

### Changes disappeared
- Make sure you clicked "Save Assignments" for each event
- Changes aren't saved until you click the save button
- If you refresh page before saving, changes are lost
