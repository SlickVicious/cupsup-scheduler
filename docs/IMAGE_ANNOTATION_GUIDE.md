## Color Scheme Recommendations:

**Arrows:** Bright red (#FF0000) with white outline
**Boxes:** Yellow (#FFFF00) with 50% transparency
**Text:** White text on semi-transparent black background
**Numbers (‚ë†‚ë°‚ë¢‚ë£):** White on bright blue circles

## Tool Recommendations:

### Free Options:

- **Preview (Mac):** Open image ‚Üí Tools ‚Üí Annotate
- **Paint 3D (Windows):** Built-in annotation tools
- **Google Slides:** Import image, add shapes/text
- **PowerPoint:** Same as Google Slides

### Professional:

- **Photoshop:** Advanced annotation
- **Figma (Free):** Web-based design tool
- **Canva (Free):** Easy drag-and-drop

---

## Quick PowerPoint Method:

1. **New blank presentation**
2. **Insert ‚Üí Pictures** (add screenshot)
3. **Insert ‚Üí Shapes** for arrows/boxes
4. **Insert ‚Üí Text Box** for labels
5. **Format ‚Üí Shape Fill** (yellow, 50% transparency)
6. **Right-click ‚Üí Save as Picture** when done

---

## Dimensions:

Keep text readable:

- **Minimum font size:** 14pt
- **Arrow thickness:** 3-4px
- **Box border:** 2px
- **Padding:** 5px around text
